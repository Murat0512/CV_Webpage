# CV Webpage

A simple, responsive CV/Resume webpage built with HTML, CSS, and JavaScript.

## Features
- Clean CV layout
- Responsive styling
- Optional interactive enhancements via JavaScript

## Project Structure
- index.html: Main markup for the CV page
- styles.css: Styles for layout and visuals
- script.js: Optional interactivity

## Getting Started
1. Open index.html in your browser.
2. (Optional) Edit the content in index.html to personalize your CV.
3. Adjust styles in styles.css as needed.

## Customization
- Update sections such as experience, education, skills, and contact details in index.html.
- Tweak colors, fonts, and spacing in styles.css.
- Add any extra interactions in script.js.
